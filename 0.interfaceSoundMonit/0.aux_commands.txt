Install mbutil:
You can clone the mbutil repository from GitHub and use it to extract the tiles from an MBTiles file.


git clone https://github.com/mapbox/mbutil.git
cd mbutil


Download or Create an MBTiles File:
You can download MBTiles from sources like OpenMapTiles. 
Make sure to choose the area and zoom levels you need for your application.

Extract the Tiles:
Use mbutil to extract the tiles from the MBTiles file into a directory structure.


./mb-util --image_format=png yourfile.mbtiles output-directory
